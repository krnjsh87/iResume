# iResume

**Professional Resume Builder - Fully Offline Desktop Application**

Create ATS-optimized resumes with support for 50+ countries and 10 professional themes.

## Features

- ✅ **50+ Country Formats**: Supports resume standards from countries across all continents
- ✅ **10 Professional Themes**: Modern, Creative, Minimalist, Executive, and more
- ✅ **Live Preview**: Real-time preview as you type
- ✅ **ATS Score**: Get instant feedback with detailed improvement suggestions (target >90%)
- ✅ **Fully Offline**: No internet connection required, no login needed
- ✅ **Resizable Panels**: Customize your workspace with draggable panel separator
- ✅ **PDF Export**: Download professional PDFs matching your preview exactly

## Quick Start

### Installation

```bash
# Install dependencies
npm install

# Start the application
npm start
```

### Building

```bash
# Build for Windows
npm run build
```

## Usage

1. **Select Country**: Choose your target country from the dropdown (sorted by continent)
2. **Select Theme**: Pick a professional theme that matches your style
3. **Fill Resume**: Complete all sections in the left panel
4. **Preview**: Watch real-time updates in the right panel
5. **Check ATS Score**: Click the score badge to see improvement suggestions
6. **Download**: Export your resume as a professional PDF
